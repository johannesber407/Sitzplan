# README
Das ist ein Kleines Projekt, um Sitzpläne in Schulklassen zu berechnen. 

## Handhabung
1. Erstelle eine Klasse - füge Schüler hinzu. Klassen können als Datei abgespeichert und aus einer Datei geladen werden.
2. Gib die Wünsche der Schüler ein. Es sind max. 5 Wünsche Pro Schüler zulässig. Die Reihenfolge spielt keine Rolle.
3. (optional) Sperre bestimmte Kombinationen (z.B. Schüler A soll nicht neben Schüler B sitzen) oder erzwinge bestimmte Kombinationen (Schüler A muss neben Schüler B sitzen).
4. Wähle die Art der Tisch Anordnugn (Standard sind 2er Paare).
5. Berechne den Sitzplan. Der Sitzplan wird durch eine Monte-Carlo Methode gewählt, und berücksichtigt dabei die vorher eingegebenen Wünsche und Kombindationen. Die Anzahl Iterationen ist wählbar. Man kann isch auch eine Zufällige Kombination ausgeben lassen, die Wünsche nicht beachtet.
